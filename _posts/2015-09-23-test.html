11. DOM操作。怎样添加、移除、移动、复制、创建和查找节点

添加：

//  原生

ele1.appendChild(ele2)  //  将ele2元素插入到ele1子元素的最后一位（父子）

parent.insertBefore(ele, parent.childNode[0])  //  将ele元素插入到parent的某个子元素的后面（兄弟）

替换:

parent.replaceChild(newNode, oldNode)     //用newNode替代oldNode，找共同父元素作为方法的对象

移除：

ele1.parentNode.removeChild(ele1)    //移除ele1元素

移动：

parent.insertBefore(node, refNode);

复制：

ele2 = ele1.cloneNode(true);//ele1复制并赋值给ele2，第二个参数为true时同时复制子元素，false时不复制子元素，要注意元素里面的字符串也是textnode节点

创建：

document.create(标签名)

查找：

document.getElementById()/getElementsByTagName()/querySelector()/querySelectorAll()
