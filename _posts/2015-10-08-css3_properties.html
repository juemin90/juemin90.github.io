3. 文本效果

hanging-punctuation 规定标点字符是否位于线框之外。 3
punctuation-trim 规定是否对标点字符进行修剪。 3
text-align-last 设置如何对齐最后一行或紧挨着强制换行符之前的行。 3
text-emphasis 向元素的文本应用重点标记以及重点标记的前景色。 3
text-justify 规定当 text-align 设置为 "justify" 时所使用的对齐方法。 3
text-outline 规定文本的轮廓。 3
text-overflow 规定当文本溢出包含元素时发生的事情。 3
text-shadow 向文本添加阴影。 3
text-wrap 规定文本的换行规则。 3
word-break 规定非中日韩文本的换行规则。 3
word-wrap 允许对长的不可分割的单词进行分割并换行到下一行。

5. transform

转换方式

1) rotate 旋转 例如，transform:rotate(45deg);

2) scale 放大，例如，transform:scale(2, 0.5); //分别对应x, y轴

3) translate 移动，例如，transform:translate(100px, -50px); //分别对应x, y轴

4) skew 扭曲，transform:skew(45deg, 45deg);  //分别对应x, y轴

5) matrix 矩阵变形，matrix([number],[number],[number],[number],[number],[number]);

和transition一起使用可以达到动画的效果。

 

6. transition

过渡

当设置过渡的属性变化的时候，变化会持续一段时间。

1) transition-property 设置过渡的CSS属性名字

2) transition-duration 设置过渡持续的事件

3) transition-time-function 设置速度效果的速度曲线

4) transition-delay 延时时间

transition属性的简写顺序即为：[transition-property, transition-duration, transition-time-function, transition-delay]

 

7. animation和keyframes

animation和keyframes结合起来可以根据设置关键帧来达到动画的效果。

animation: name duration timing-function delay iteration-count direction;
animation-name 规定需要绑定到选择器的 keyframe 名称。。
animation-duration 规定完成动画所花费的时间，以秒或毫秒计。
animation-timing-function 规定动画的速度曲线。
animation-delay 规定在动画开始之前的延迟。
animation-iteration-count 规定动画应该播放的次数。
animation-direction 规定是否应该轮流反向播放动画。

设置完了animation，设置@keyframes，为特定名字的animation创建关键帧。

example code:

@keyframes myfirst
{
0%   {background: red; left:0px; top:0px;}
25%  {background: yellow; left:200px; top:0px;}
50%  {background: blue; left:200px; top:200px;}
75%  {background: green; left:0px; top:200px;}
100% {background: red; left:0px; top:0px;}
}

@-moz-keyframes myfirst /* Firefox */
{
0%   {background: red; left:0px; top:0px;}
25%  {background: yellow; left:200px; top:0px;}
50%  {background: blue; left:200px; top:200px;}
75%  {background: green; left:0px; top:200px;}
100% {background: red; left:0px; top:0px;}
}

@-webkit-keyframes myfirst /* Safari 和 Chrome */
{
0%   {background: red; left:0px; top:0px;}
25%  {background: yellow; left:200px; top:0px;}
50%  {background: blue; left:200px; top:200px;}
75%  {background: green; left:0px; top:200px;}
100% {background: red; left:0px; top:0px;}
}

@-o-keyframes myfirst /* Opera */
{
0%   {background: red; left:0px; top:0px;}
25%  {background: yellow; left:200px; top:0px;}
50%  {background: blue; left:200px; top:200px;}
75%  {background: green; left:0px; top:200px;}
100% {background: red; left:0px; top:0px;}
}

frame的位置可以用百分比，也可以用from-to来表示。这样，元素就会按照frame里设置的CSS轨迹进行运动。
